## Gap

The gap or space can be created in between the horizontal and vertical elements as follows:

`gap-x-WriteNumber` : for creating gap bewteen hotizontal elements or on x axis.
`gap-y-WriteNumber` : for creating gap bewteen vertical elements or on y axis.

For Example:

<div>
  <div class="container grid grid-cols-3 gap-x-7">
     <div>
       <h2>Heading 1</h2>
       <p>Para 1</p>
     </div>
     <div>
       <h2>Heading 2</h2>
       <p>Para 2</p>
     </div>
     <div>
       <h2>Heading 3</h2>
       <p>Para 3</p>
     </div>
     <div>
       <h2>Heading 4</h2>
       <p>Para 4</p>
     </div>
     <div>
       <h2>Heading 5</h2>
       <p>Para 5</p>
     </div>
     <div>
       <h2>Heading 6</h2>
       <p>Para 6</p>
     </div>
  </div>
</div>

The Above code will create three columns and will adjust or wrap other into another row automatically with the gap of 7 (horizontal effect).
